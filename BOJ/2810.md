2810 컵홀더
=============
### 문제
* 극장 한 줄 자리가 N개
* 서로 인접한 좌석 사이에 컵홀더 하나씩 있음
* 양 끝 좌석에 컵홀더 하나씩 더 있음
* 커플석 사이에는 컵홀더가 없음
* 극장의 한 줄 정보가 주어짐 이 줄에 사람들이 모두 앉았을 때, 컵홀더에 컵을 꽂을 수 있는 최대 사람의 수
* 모든 사람은 컵을 한 개만 들고 있음
* 자신의 좌석 양 옆에 있는 컵홀더에만 컵을 꽂을 수 있음
* S는 일반 좌석, L은 커플석, L은 항상 두개씩 쌍으로 주어짐
### 입력
* 첫쨰 줄에 좌석의 수 N (1 <= N <= 50)
* 둘째 줄에 좌석 정보
### 출력
* 컵을 컵홀더에 놓을 수 있는 최대 사람의 수

접근 방법
-------------
* 커플석이 존재하는 경우와 존재하지 않는 경우로 나눔
* 커플석이 존재하지 않는다면,
    * 모두 컵홀더를 이용할 수 있음
* 커플석이 존재한다면,
    * 커플석이 나오기 전까지는 자리의 왼쪽 컵홀더 이용하고,
    * 이후에는 자리의 오른쪽 컵홀더를 이용하면
    * 모든 컵홀더가 이용 됨
